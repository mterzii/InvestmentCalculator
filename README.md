# Investment Calculator (React)

A React-based investment calculator that shows yearly investment growth,
annual interest earned, total interest, and invested capital.

## Features
- Controlled form inputs
- Yearly investment calculation
- Dynamic results table
- Currency formatting

## Tech Stack
- React
- JavaScript
- Vite
